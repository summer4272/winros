# Source Notes

These official sources shaped the initial scaffold on 2026-05-25.

- ROS developer documentation lists Lyrical as the latest LTS and shows platform support.
- ROS 2 Jazzy Windows binary installation documents the Windows baseline and `C:\pixi_ws` path note.
- ROS 2 Lyrical Windows binary installation currently describes prerelease binaries.
- MuJoCo Python documentation recommends prebuilt Python packages for ordinary use and documents the viewer entry point.
- PyTorch local installation documentation recommends choosing Windows CPU/CUDA wheels through the official selector.
- MoveIt 2 Windows binary install notes Windows binaries through RoboStack.
